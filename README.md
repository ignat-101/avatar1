# 💎 TON Avatar Generator & Telegram Bot

Это обновленная, ультра-минималистичная и стильная версия генератора аватарок для TON-доменов.

## 🌟 Что внутри?
1. **Web Preview (React + Vite)** — живое превью генерации аватарок. Вы можете поиграться с настройками (ввод домена, выбор из 8 фонов, glass-эффект, TON логотип) прямо в браузере.
2. **Telegram Bot (Node.js)** — полноценный бот с инлайн-кнопками. Генерирует и отправляет аватарки прямо в диалог. По умолчанию, если пользователь не указал домен, используется `7288.ton`.

## 🎨 Дизайн
Улучшенный дизайн опирается на принципы минимализма и Apple-подобной эстетики:
- **8 премиальных фонов** (плавные градиенты, минималистичные оттенки).
- **Glassmorphism 2.0** — улучшенное "стекло" с тонкой белой рамкой и легкой тенью.
- **Типографика** — строгий, крупный, выровненный по центру шрифт.

## 🚀 Как запустить Web-превью

Просто установите зависимости и запустите dev-сервер:
```bash
npm install
npm run dev
```
Откройте локальный адрес (обычно `http://localhost:5173`), чтобы увидеть превью аватарок.

## 🤖 Как запустить Telegram Бота

Бот находится в папке `bot`. 

### 1. Системные зависимости (для Canvas)
Убедитесь, что у вас установлены зависимости для библиотеки `canvas`.
**Ubuntu:** `sudo apt-get install -y build-essential libcairo2-dev libpango1.0-dev libjpeg-dev libgif-dev librsvg2-dev`
**Mac:** `brew install pkg-config cairo pango libpng jpeg giflib librsvg`

### 2. Настройка
Зайдите в папку бота и установите пакеты:
```bash
cd bot
npm install
```

Создайте файл `.env` в папке `bot`:
```env
BOT_TOKEN=ВАШ_ТОКЕН_ОТ_BOTFATHER
```

### 3. Запуск
```bash
npm start
```

### Как пользоваться ботом?
- Напишите `/start`
- Используйте `/avatar` (сгенерирует для 7288.ton) или `/avatar myname.ton`
- Под картинкой появятся инлайн-кнопки для переключения фона, стекла и логотипа.

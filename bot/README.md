# TON Avatar Bot 🤖

Telegram бот для генерации аватаров прямо в диалоге — без TMA (Telegram Mini App).

## Возможности

- 🎨 Генерация аватаров 1024×1024 PNG
- 8 тем оформления (TON Blue, Neon Cyber, Deep Space и др.)
- Glassmorphism эффект
- TON логотип (outlined, как на официальном сайте TON)
- Инлайн-кнопки для выбора темы и настроек
- Автоматическое запоминание имени из текстовых сообщений

## Установка

```bash
cd bot
npm install
```

## Настройка

Создайте файл `.env` в папке `bot/`:

```
BOT_TOKEN=ваш_токен_от_@BotFather
```

## Запуск

```bash
node index.js
# или для разработки:
npx nodemon index.js
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск и главное меню |
| `/avatar` | Создать аватар с текущими настройками |
| `/avatar имя.ton` | Создать аватар с указанным именем |
| `/random` | Случайная тема + генерация |
| `/themes` | Выбрать тему |
| `/settings` | Настройки (glassmorphism, лого, форма) |
| `/help` | Справка |

## Инлайн-кнопки

Все действия доступны через кнопки — никаких дополнительных команд вводить не нужно.

## Изменение имени

Просто напиши любой текст в чат (не команду), и бот запомнит его как имя для следующей генерации.

Пример: `vitalik.ton`

## Зависимости

- [telegraf](https://telegraf.js.org/) — Telegram Bot Framework
- [canvas](https://www.npmjs.com/package/canvas) — генерация изображений на сервере
- [dotenv](https://www.npmjs.com/package/dotenv) — переменные окружения

# TON Avatar Bot

Telegram бот для генерации стильных аватарок для TON доменов.

## 🚀 Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ignat-101/avatar1.git
   cd avatar1/bot
   ```

2. Установите зависимости:
   ```bash
   npm install telegraf canvas dotenv
   ```

3. Настройте переменные окружения:
   Создайте файл `.env` в папке `bot` и добавьте туда токен вашего бота:
   ```env
   BOT_TOKEN=ваш_токен_от_BotFather
   ```

4. Запустите бота:
   ```bash
   node index.js
   ```

## 🛠 Исправления

В текущей версии устранены следующие проблемы:
1. **Черные аватары**: Исправлено поведение `node-canvas` с прозрачным фоном, который Telegram конвертировал в черные JPEG. Добавлен сплошной белый фон перед отрисовкой градиента, а также формат отдачи изменен на `image/jpeg`.
2. **Неработающие кнопки настроек**: Функция `settingsKeyboard` отсутствовала, из-за чего бот крашился при нажатии на "Настройки". Она была добавлена, и теперь смена параметров (Форма, Глассморфизм, Логотип) работает корректно.
3. **Missing Generation Logic**: Функция `sendAvatar` не была определена в оригинальном коде, что ломало генерацию. Она добавлена и корректно оборачивает вызов функции с обработкой ошибок.
4. **Themes Keyboard Error**: Исправлена ошибка неверного формата в функции `themesKeyboard`.
